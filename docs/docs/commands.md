# 命令

```bash
帮助 —— XWebServer+
python server.py <操作> [<选项>...]

    help：帮助
    about：关于
    create：服务器创建向导
    start：开启服务器
        -c <path>   指定服务器配置所在路径
```

## 操作

### HELP
查看帮助

### ABOUT
关于 XWebServer+

### CREATE
打开服务器创建向导

### START
开启服务器

#### -C &lt;path&gt;
指定服务器配置目录
